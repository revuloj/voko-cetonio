Uzo de personaj informoj

Redaktilo por Reta Vortaro bezonas uzi kelkajn personajn informojn por ensalutado kaj subskribado de senditaj redaktoj:

  • Vi povas ensaluti per konto ĉe alia retprovizanto: Google, Facebook, Yahoo. Redaktilo uzas vian nomon kaj se eble retadreson el tiu konto por rekoni vin kaj permesi ensalutadon al la aplikaĵo.
  • Per uzado de via konto Yahoo, Facebook aŭ Google ekscias, ke vi uzas la servon ĉe revaj.steloj.de
  • Se la retadreso de la salutokonto ne estas uzebla, Redaktilo demandas vin pri via retadreso.
  • Via retadreso estas uzata por subskribi redaktojn senditajn al Revo-Redaktoservo. Per tiu retadreso la redaktoservo rekonas vin kiel redaktanto kaj subskribas la redakton per via nomo registrita kun tiu retadreso. Redaktoj ankaŭ sendiĝas kopie al tiu via retadreso.

Vi povas ekscii pli pri kiel Google uzas viajn datumojn ĉe: http://www.google.com/intl/eo/policies/privacy.

Vi povas ekscii pli pri kiel Facebook uzas viajn datumojn ĉe: https://www.facebook.com/privacy/explanation

Vi povas ekscii pli pri kiel Yahoo uzas viajn datumojn ĉe: https://policies.yahoo.com/ie/en/yahoo/privacy.

Uzo de kuketoj

Redaktilo por Revo uzas la sekvajn kuketojn:
  • Redaktilo uzas seanco-kuketon (redaktilo_seanco), tio estas simpla nedivenebla numera identigilo, per kiu la aplikaĵo povas identigi vin dum redaktado. Necesas memori vian identecon por scii, ke vi antaŭe salutis. La seanco-kuketo forgesiĝas kiam vi fermas vian krozilon.
  • Aldone kreiĝas dua kuketo (redaktilo_ajax) el identiga numero interne ligita kun via nomo kaj retadreso (sed ĝi mem ne enhavas vian retadreson kaj nomon). La identiga numero estas signita per sekreto, tiel ke sen scii la sekreton, ĝi ne povas esti falsata. Ĝi estas uzata por aŭtentigi la fonajn datumricevojn, ekz. dum serĉo, antaŭrigardo, artikolkontrolo kaj forsendo de artikolo kaj ĝi validas kelkajn tagojn tiel, ke vi povas plurajn tagojn redakti sen resaluti, se vi ne fermas la krozilon.
  • Redaktilo uzas kiel Javoskripto-bibliotekoj jQuery kaj jQuery UI. Tiuj bibliotekoj serviĝas de speciala enhav-serva reto CloudFlare por rapida preno el code.jquery.com, kiu ankaŭ kreas kelkajn kuketojn en via krozilo, i.a. de Google Analytics.

Via konsento

Sen la supre menciitaj informoj Redaktilo ne povas funkcii por redakti artikolojn de Revo kaj sendi ilin kun subskribo.

Per ensalutado kaj uzo de Redaktilo vi konsentas uzi viajn personajn informojn kiel supre priskribita.
